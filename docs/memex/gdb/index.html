GDB Commands

  • start gdb:
    • gdb
  • select file:
    • file a.out
  • execute file:
    • run
  • open source code window:
    • layout next
  • set breakpoint on function-name invocaion:
    • b function-name
  • next line:
    • next or n
  • print var-name:
    • p var-name
  • update me when var value changes:
    • watch var-name
  • step in function:
    • s
  • continue execution until next breakpoint:
    • continue or c
